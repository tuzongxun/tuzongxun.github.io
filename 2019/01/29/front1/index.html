еүҚз«Ҝ

vue+element-uiжҗӯе»әзәҜеүҚеҸ°йЎ№зӣ®и®°еҪ•

жһ¶жһ„иҜҙжҳҺ

жң¬ж¬ЎйЎ№зӣ®з”ЁдәҺдёӘдәәеӯҰд№ пјҢдҪңз”ЁдәҺдёӘдәәиҒҡеҗҲж”Ҝд»ҳdemoпјҢи®°еҪ•жӯҘйӘӨпјҢдёәд»ҘеҗҺдҪңеҸӮиҖғгҖӮ
еүҚеҸ°йЎ№зӣ®жҗӯе»әзҡ„жһ¶жһ„еҹәзЎҖжҳҜеүҚеҗҺеҸ°еҲҶзҰ»пјҢеҚіпјҡд»Һд»Јз ҒеұӮйқўжқҘиҜҙпјҢеүҚеҸ°е’ҢеҗҺеҸ°дә’дёҚзӣёе№ІпјҢдёҚеҗҢзҡ„жңҚеҠЎпјҢдёҚеҗҢзҡ„з«ҜеҸЈпјҢеүҚеҗҺеҸ°д№ӢеүҚдҪҝз”ЁhttpеҚҸи®®иҝӣиЎҢдәӨдә’гҖӮ
еүҚеҸ°еҹәжң¬жҠҖжңҜжһ¶жһ„дёәnode.js+vue.js+element-ui+vue-routerгҖӮ

жӯҘйӘӨиҜҙжҳҺ

е®үиЈ…node.js

жң¬йЎ№зӣ®еҹәдәҺnode.jsжҗӯе»әпјҢйңҖиҰҒдҪҝз”Ёnpmе‘Ҫд»ӨпјҢз”ұдәҺжҲ‘д№ӢеүҚе·Із»Ҹе®үиЈ…иҝҮпјҢеӣ жӯӨи·іиҝҮгҖӮ

еҲӣе»әvueйЎ№зӣ®

vueеҲқе§ӢйЎ№зӣ®жҗӯе»әиҝҮзЁӢдё»иҰҒеҸӮиҖғеҰӮдёӢеҚҡе®ўпјҡ
еүҚз«Ҝжһ¶жһ„д№Ӣи·ҜпјҡдҪҝз”ЁVue.jsејҖе§Ӣз¬¬дёҖдёӘйЎ№зӣ®
зЁҚжңүдёҚеҗҢзҡ„жҳҜпјҢдёҠиҫ№еҚҡж–ҮдёӯдҪҝз”Ёзҡ„жҳҜйҳҝйҮҢзүҲnodeпјҢжүҖд»ҘжҳҜcnpmе‘Ҫд»ӨпјҢиҖҢжҲ‘з”Ёзҡ„жҳҜеҺҹзүҲпјҢжүҖд»Ҙз”Ёзҡ„жҳҜnpmгҖӮ
з®ҖеҚ•ж•ҙзҗҶдёҖдёӢжңҖз»ҲжӯҘйӘӨпјҢеӨ§иҮҙеҰӮдёӢпјҡ

е®үиЈ…webpack

зҪ‘дёҠиҜҙwebpackжҳҜдёҖдёӘжЁЎжқҝжү“еҢ…еҷЁпјҢе®һйҷ…дҪҝз”Ёж—¶жҲ‘и§үеҫ—дјјд№ҺиҜҙжҳҜдёҖдёӘжһ„е»әе·Ҙе…·жӣҙеҗҲзҗҶпјҢдҪҝз”ЁеҰӮдёӢе‘Ҫд»Өе®үиЈ…пјҡnpm install -g webpack

е®үиЈ…vue-cli

дҪҝз”Ёnpm install vue-cli -gе‘Ҫд»Өе®үиЈ…и„ҡжүӢжһ¶пјҢзӣ®еүҚзҗҶи§Јдёәе°ұжҳҜдёҖдёӘеҲӣе»әvueйЎ№зӣ®зҡ„е·Ҙе…·гҖӮ

еҲӣе»әvueйЎ№зӣ®

дҪҝз”Ёvue-cliзҡ„vueе‘Ҫд»ӨеҲӣе»әйЎ№зӣ®жЁЎжқҝпјҢиҝҷйҮҢе®һйҷ…дёҠиё©дәҶдёӘеқ‘гҖӮдёҠиҫ№еҚҡж–ҮдёӯеҲӣе»әе‘Ҫд»ӨжҳҜvue init webpack-simple first-vueпјҢжҲ‘жҳҜз…§зқҖж•Ізҡ„пјҢдёҖејҖе§ӢжІЎжңүй—®йўҳпјҢдҪҶжҳҜеҲ°еҗҺиҫ№дҪҝз”Ёvue-routerи·Ҝз”ұзҡ„ж—¶еҖҷеҗ„з§Қй—®йўҳеҮәзҺ°гҖӮеҺҹеӣ е°ұжҳҜwebpack-simpleжҗӯе»әзҡ„жҳҜжһҒз®ҖзүҲйЎ№зӣ®пјҢеҰӮжһңиҰҒдҪҝз”ЁrouterиҝҷдәӣеҠҹиғҪпјҢд»Һдҫқиө–еҲ°зӣ®еҪ•йғҪиҰҒиҮӘе·ұдёҖдёӘдёӘзҡ„еҲӣе»әгҖӮеҜ№дәҺеҲҡжҺҘи§Ұvueзҡ„дәәжқҘиҜҙпјҢиҝҷдёӘиҝҮзЁӢеҝ…з„¶йў‘йў‘еҮәй”ҷпјҢжүҖд»ҘеҗҺжқҘеҫ—зҹҘжңҖдҪіжӯҘйӘӨеә”иҜҘжҳҜиҝҷж ·vue init webpack first-vue

еј•е…Ҙelement-uiгҖӮ

дҪҝз”Ёelement-uiпјҢе…ҲиҰҒеңЁйЎ№зӣ®дёӯе®үиЈ…element-uiзҡ„дҫқиө–пјҢеңЁйЎ№зӣ®ж №зӣ®еҪ•дёӢжү§иЎҢеҰӮдёӢе‘Ҫд»Өnpm install element-ui -SгҖӮ
з„¶еҗҺйңҖиҰҒеңЁйЎ№зӣ®main.jsдёӯеҠ е…Ҙиҝҷж ·еҮ иЎҢпјҡ

import ElementUI from 'element-ui';
import 'element-ui/lib/theme-chalk/index.css';
Vue.use(ElementUI);

жіЁж„ҸдәӢйЎ№

webpackе’Ңwebpack-simpleзҡ„еҢәеҲ«

иҝҷдёӘеңЁдёҠиҫ№иҜҙиҝҮпјҢдҪҝз”ЁwebpackдјҡеңЁйЎ№зӣ®дёӯеҲӣе»әеҘҪи·Ҝз”ұзӣёе…ізҡ„з»“жһ„пјҢиҖҢwebpack-simpleдёҚдјҡгҖӮ

дҫқиө–еҠ иҪҪ

дёҚз®ЎжҳҜеҲқе§ӢйЎ№зӣ®еҗҺпјҢиҝҳжҳҜеҗҺиҫ№еҠ е…Ҙelement-uiд№ӢеҗҺпјҢйғҪжҳҜйңҖиҰҒдҪҝз”Ёзұ»дјјnpm installиҝҷж ·зҡ„е‘Ҫд»Өе®үиЈ…еҠ иҪҪдҫқиө–зҡ„пјҢе°ұеҰӮjava mavenйЎ№зӣ®дёӯimportдёҖдёӘзұ»жҲ–иҖ…еҢ…пјҢеҝ…йЎ»иҰҒmavenеҠ иҪҪдәҶзӣёеә”зҡ„jarеҢ…жүҚиЎҢгҖӮ

еёёи§„зҡ„vueж–Үд»¶еӨ§жҰӮжҳҜзұ»дјјдёӢиҫ№зҡ„ж јејҸпјҡ

<template>
  <div id="myvue"></div>
</template>
<script>
</script>

<style>

иҝҷдёӘжЁЎжқҝеҰӮжһңиҰҒдәӨз»ҷrouterи·Ҝз”ұз®ЎзҗҶпјҢе°ұйңҖиҰҒеңЁи·Ҝз”ұдёӯй…ҚзҪ®пјҢеҰӮrouterзӣ®еҪ•дёӢзҡ„index.jsдёӯй…ҚзҪ®пјҢдҫӢеҰӮпјҡ
import myvue from '@/components/myvue'
е…¶дёӯпјҢimportеҗҺиҫ№зҡ„myvueжҳҜдёҠиҫ№зҡ„idзҡ„еҖјпјҢcomponentsеҗҺиҫ№зҡ„myvueжҳҜcomponentsзӣ®еҪ•дёӢзҡ„myvue.vueж–Үд»¶еҗҚгҖӮ
з„¶еҗҺй…ҚзҪ®и·Ҝз”ұз»„д»¶,еңЁroutesдёӯеҠ е…Ҙзұ»дјјдёӢиҫ№иҝҷж ·зҡ„д»Јз Ғпјҡ

  name: 'myvue',
